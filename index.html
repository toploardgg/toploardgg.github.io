Get started

Join in
four steps.

  1. 1

    Launch Minecraft Java Edition

    Open the Minecraft Java launcher and sign in with your Microsoft or Mojang account.

  2. 2

    Go to Multiplayer

    From the main menu click "Multiplayer", then "Add Server" at the bottom of the list.

  3. 3

    Enter the server address

    Paste the IP address from the card on the right into the Server Address field, then click Done.

  4. 4

    Select and connect

    Select Toploardgg from your server list and click Join Server. Welcome to the community.
